Ejercicio de encabezados HTML

Objetivo:

El objetivo de este ejercicio es que aprendas a usar las etiquetas de encabezado HTML para estructurar el contenido de una página web.

Más información sobre encabezados en HTML AQUÍ

Lo primero que debes hacer es FORK de este documento 💾 Recuerda guardar tus cambios para no perderlos.

Instrucciones:

En el cuerpo del documento, agrega los siguientes encabezados:

  • Un encabezado de nivel 1 con el texto "Título de la página"
  • Un encabezado de nivel 2 con el texto "Sección 1"
  • Un encabezado de nivel 3 con el texto "Subsección 1"
  • Un encabezado de nivel 4 con el texto "Subsubsección 1"
  • Un encabezado de nivel 5 con el texto "Subsubsubsección 1"
  • Un encabezado de nivel 6 con el texto "Subsubsubsubsección 1"

Entre cada encabezado, agrega un párrafo de texto con contenido informativo, con una descripción sobre la utilidad de ese encabezado.
